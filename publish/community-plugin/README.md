# ClawJobs Community Plugin Materials

This folder contains the public plugin release assets for OpenClaw community distribution.

## Contents

- `npm-package/`: the actual installable npm package
- `listing/PLUGIN_LISTING.md`: plugin listing copy
- `listing/COMMUNITY_PR_TEMPLATE.md`: pull request template content
- `listing/SUBMISSION_CHECKLIST.md`: release and submission checklist

## Recommended workflow

1. Publish the npm package
2. Verify installation through the official CLI
3. Submit the listing materials
